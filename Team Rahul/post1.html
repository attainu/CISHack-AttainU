Food Delivery App

This was a project I started as my very first frontend project when I first learned HTML, CSS AND JavaScript. I was the sole creator/developer for the of the project.

The system stored sample data for the restraunts search and provided all the data about a particular restraunt and admins to view, add and send data to clients.

search results would be generated by the fetching the data from the API and the sytem had location based filters and category based filters interface for customers to be able to login and view.

Customers had the ability to view all their data with a dynamic search form, trend data.
